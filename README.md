# Vilgots Garderob

Statisk lookbook med gentleman-babykläder (kostym, smoking, fluga, hängslen).

Publicerad via GitHub Pages: https://mikbol.github.io/vilgot-garderob/

## Filer

- `index.html` — sidan + produktdata (inline JSON)
- `style.css` — styling
- `app.js` — rendering, filter, localStorage
- `img/` — produktbilder

## Lokal utveckling

Öppna `index.html` direkt i webbläsaren, eller kör en enkel server:

```sh
python3 -m http.server 8000
```

## Deploy

Pusha till `main` — GitHub Pages uppdaterar sig automatiskt.
